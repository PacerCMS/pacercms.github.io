We are proud to announce the availability of PacerCMS 0.5 PR1. This preview release gives you the opportunity to start using the new template engine and installer included with PacerCMS. As with most open source software, we have a few groups we would like to thank.

  • The WordPress developers for creating a powerful yet simple installer that our version could "clone." If you look closely, you will find other WordPress-inspired tools (JS Quick Tags and autop(), for example) littered throughout the code.
  • The Smarty developers for a template engine that makes customizing PacerCMS quite a bit easier without worrying about breaking the PHP code.
  • You, for clicking the download link below and giving it a spin.

Let us know what you think of the "new stuff" in our PacerCMS Developers Group. This download package will not count toward our total number, so your feedback is really important to us.
