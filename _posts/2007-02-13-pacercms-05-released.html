A few days ago, we committed the latest stable version of PacerCMS. Thank you to all those who tried out our Preview Releases and for the much needed feedback. Version 0.5 marks the biggest step forward for the system yet, introducing four great new features that we have highlighted in past entries. We have a growing list of features for 0.6 on our Development page, so please take a look and add your feature requests.

This Release (Subversion R65):

  • The Smarty Template Engine (#4)
  • No Hassle Installation Script (#3)
  • JavaScript Quick Tags for Articles (#6)
  • Flash Media For Articles
